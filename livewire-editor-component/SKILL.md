    Build Livewire Editor components for editing complex content with multiple
    related parts. Coordinates several forms that work together with a single save

    editors, or cohesive data entry tasks.

## When to use me

- Creating a workspace for editing complex content
- Multiple forms on one screen that serve the same purpose
- Everything exists to edit one cohesive concept
- Need a single save action that persists multiple related forms

- Follow architecture priority from `naykel-laravel`.
